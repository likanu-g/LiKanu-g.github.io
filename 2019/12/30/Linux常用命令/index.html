Linux的常用命令

  • 帮助命令

    • help
      • 功能：查看 Shell 内部命令的帮助信息
      • 语法：help (选项)(参数)
      • 示例：help //可以不带参数，查看所有内置命令的帮助预览
        help for  //查看for的用法
    • whatis
      • 功能：查询一个命令执行什么功能
      • 语法：whatis [命令]
      • 示例：whatis ls //查看ls命令的功能
    • info
      • 功能：info 格式的帮助指令
      • 语法：info (选项)(参数)
      • 示例：info ls //显示info格式的ls命令的文档
    • which
      • 功能：用于查找并显示给定命令的绝对路径
      • 语法：which (选项)(参数)
      • 示例： which pwd //查看pwd命令所在的路径
    • whereis
      • 功能：用来定位指令的二进制程序、源代码文件和man手册页等相关文件的路径
      • 语法：whereis (选项)(参数)
      • 示例：whereis java //查看java的路径
    • man
      • 功能：功能和info基本一致，只是显示的帮助信息格式不一样
      • 语法：man (选项)(参数)
      • 示例：man java //查看java的帮助文档

  • 文件目录管理

    • 目录管理

      • cd

        • 功能：切换目录
        • 语法：cd (选项) (参数)
        • 示例：cd ~ //进入用户主目录
          cd ../../   //切换上两级目录

      • ls

        • 功能：列出目录内容
        • 语法：ls (选项)(参数
        • 示例：ls -l //以列表形式显示当前目录下的内容

      • pwd

        • 功能：显示当前目录下的绝对路径

        • 语法：pwd (选项)

        • 示例：pwd

          ​ pwd引用当前目录的路径，可用于集群服务器间的远程分发

      • mkdir

        • 功能：创建目录
        • 语法：mkdir (选项)(参数)
        • 示例：mkdir test //当前目录下创建test文件夹
          mkdir -m 700 /usr/meng/test  //创建权限为700的文件夹test,在/usr/meng下面

      • rmdir

        • 功能：删除目录
        • 语法：rmdir (选项)(参数)
        • 示例：rmdir test //删除当前目录下名为test的文件夹

      • tree

        • 功能：树状显示目录的内容
        • 语法：tree (选项)(参数)
        • 示例：tree -a //以树形格式显示所有目录

    • 文件管理

      • touch
        • 功能：创建空文件
        • 语法：touch (选项)(参数)
        • 示例：touch test.txt //创建text.txt的文本文件
          touch -m test.txt    //更改test.txt的修改时间
      • ln
        • 功能：为文件创建链接(windows中的快捷方式)
        • 语法：ln (选项)(参数)
        • 示例：ln /mub1/m2.c /usr/liu/a2.c //给m2.c创建一个链接到a2c
      • rename
        • 功能：批量重命名
        • 语法：rename(参数)
        • 示例：rename test.txt rest.a test.txt
      • stat
        • 功能：显示文件的详细信息
        • 语法：stat(选项)(参数)
        • 示例：stat test.txt //显示test.txt的详细信息(相较于ls而言)
      • file
        • 功能：检测文件类型
        • 语法：file(选项)(参数)
        • 示例：file jdk-8u221-linux-x64.tar.gz //检测这个jdk的文件类型
      • chmod
        • 功能：设置文件或者目录的权限
        • 语法：chmod(选项)(参数)
        • 示例：chmod 777 test.txt //把test.txt文件的权限改成可读写和执行 
          chmod +x test.txt       //给test.txt添加可执行权限
      • chown
        • 功能：设置文件或目录的拥有者或所属群组
        • 语法：chown(选项)(参数)
        • 示例：chown -R liu /usr/meng //将目录/usr/meng及其下面的所有文件、子目录的文件主改成 liu：
      • locate
        • 功能：查找文件或者目录
        • 语法：locate/slocate(选项)(参数)
        • 示例：locate /etc/sh //搜索etc目录下所有以sh开头的文件
      • find
        • 功能：在指定目录下查找文件
        • 语法：find(选项)(参数)
        • 示例：find /home -name “*.txt” //在/home目录下查找以.txt结尾的文件名

    • 文件目录通用

      • cp
        • 功能：复制文件或者目录
        • 语法：cp(选项)(参数)
        • 示例：cp file /usr/men/tmp/file1 //将文件file复制到目录/usr/men/tmp下，并改名为file1
      • mv
        • 功能：移动文件或者目录
        • 语法：mv(选项)(参数)
        • 示例：mv test.txt app.txt //将test.txt改名为app.txt
      • rm
        • 功能：删除文件或者目录
        • 语法：rm (选项)(参数)
        • 示例：rm -f test.txt //强制删除test.txt文件，没有提示

  • 文件内容查看

    • cat
      • 功能：经常用来显示文件的内容
      • 语法：cat(选项)(参数)
      • 示例：cat m1 （在屏幕上显示文件ml的内容）
        cat m1 m2 （同时显示文件ml和m2的内容）
cat m1 m2 > file （将文件ml和m2合并后放入文件file中）
    • head
      • 功能：用于显示文件的开头的内容。在默认情况下，head命令显示文件的头10行内容
      • 语法：head(选项)(参数)
      • 示例：head test.txt //显示test.txt的前10行内容
    • tail
      • 功能：用于输入文件中的尾部内容。tail命令默认在屏幕上显示指定文件的末尾10行
      • 语法：tail(选项)(参数)
      • 示例：tail -n 5 rest.a //显示rest.a后5行的数据
    • more
      • 功能：是一个基于vi编辑器文本过滤器，它以全屏幕的方式按页显示文本文件的内容，支持vi中的关键字定位操作
      • 语法：more(选项)(参数)
      • 示例：more -c -10 file //显示文件file的内容，每10行显示一次，而且在显示之前先清屏
    • less
      • 功能：浏览文字档案的内容功能和more基本一致，既可以向前翻也可以向后翻页
      • 语法：less(选项)(参数)
      • 示例：less test.txt //查看test.txt文件
    • sed
      • 功能：sed 是一种流编辑器，它是文本处理工具
      • 语法：sed [options] ‘command’ file(s)
      • 示例：sed ‘5a我是’ rest.a //在第五行追加“我是” ，5表示第5行，a表示追加
    • vi
      • 功能：文本编辑器类比windows的记事本
      • 语法：vi aa.c //如果没有aa.c文件则创建这样一个文件，如果有就打开这个文件
      • 示例：
    • grep
      • 功能：一种强大的文本搜索工具，它能使用正则表达式搜索文本，并把匹配的行打印出来
      • 语法：grep match_pattern filename
      • 示例：grep 32 rest.a //查找rest.a文件中包含32的行

  • 文件压缩解压

    • tar
      • 功能:为 linux 的文件和目录创建档案，常用于解压缩文件
      • 语法：tar(选项)(参数)
      • 示例:tar -cvf log.tar test.log //仅打包，不压缩！ 
        tar -zcvf log.tar.gz test.log   //打包后，以 gzip 压缩
tar -ztvf log.tar.gz           //查看log.tar.gz下有哪些文件
tar -zxvf log.tar.gz           // 解压缩log.tar.gz
    • gzip
      • 功能:gzip是个使用广泛的压缩程序，文件经它压缩过后，其名称后面会多处“.gz”扩展名
      • 语法：gzip(选项)(参数)
      • 示例:gzip log.tar //压缩log.tar文件
    • zip
      • 功能:可以用来解压缩文件，或者对文件进行打包操作
      • 语法：zip(选项)(参数)
      • 示例:zip -q -r html.zip html //将html文件夹下的所有文件压缩成html.zip
    • unzip
      • 功能:用于解压缩由zip命令压缩的“.zip”压缩包
      • 语法：unzip(选项)(参数)
      • 示例:unzip test.zip //将压缩文件text.zip在当前目录下解压缩

  • 用户管理

    • useradd
      • 功能：用于Linux中创建的新的系统用户
      • 语法：useradd(选项)(参数)
      • 示例：useradd admin -u 666 //添加一个admin用户用户id为666
    • userdel
      • 功能：用于删除给定的用户，以及与用户相关的文件。若不加选项，则仅删除用户帐号，而不删除相关文件
      • 语法：userdel(选项)(参数)
      • 示例：userdel -r admin //删除用户admin，其家目录及文件一并删除；
    • usermod
      • 功能：用于修改用户的基本信息。usermod命令不允许你改变正在线上的使用者帐号名称。当usermod命令用来改变user id，必须确认这名user没在电脑上执行任何程序
      • 语法：usermod(选项)(参数)
      • 示例：usermod -l newuser1 newuser //修改newuser1 的用户名为newuser
    • passwd
      • 功能：用于设置用户的认证信息，包括用户密码、密码过期时间等
      • 语法：passwd(选项)(参数)
      • 示例：passwd //修改当前用户的密码
    • su
      • 功能：用于切换当前用户身份到其他用户身份,变更时须输入所要变更的用户帐号与密码
      • 语法：su(选项)(参数)
      • 示例：su -admin //切换成admin账号
    • sudo
      • 功能：用来以其他身份来执行命令，预设的身份为root
      • 语法：sudo(选项)(参数)
      • 示例：

  • 系统管理

    • reboot
      • 功能：重启系统
    • exit
      • 功能：退出hsell，并返回给定值
    • shutdown
      • 功能：关闭系统
    • date
      • 功能：查看或设置系统时间与日期
    • mount
      • 功能：挂在文件系统
    • umount
      • 功能：取消挂载文件系统
    • ps
      • 功能：查看系统当前进程状态
    • kill
      • 功能：删除当前正在运行的程序
    • systemctl
      • 功能：启动、停止、重启、关闭、显示系统服务（Centos7）
    • service
      • 功能：启动、停止、重启、关闭、显示系统服务（Centos7 以前）
    • crontab
      • 功能：管理需要周期性执行的任务

  • 网络管理

    • curl

      • 功能：一个利用URL规则在命令行下工作的文件传输工具。它支持文件的上传和下载
      • 语法：curl(选项)(参数)
      • 示例：curl http://man.linuxde.net/test.iso -o filename.iso –progress //

    • wget

      • 功能：用来从指定的URL下载文件
      • 语法：wget(选项)(参数)
      • 示例：wget http://www.linuxde.net/testfile.zip

    • telnet

      • 功能：用于登录远程主机，对远程主机进行管理
      • 语法：telnet(选项)(参数)
      • 示例：telnet 192.168.2.10

    • ip

      • 功能：用来显示或操纵Linux主机的路由、网络设备、策略路由和隧道，是Linux下较新的功能强大的网络配置工具
      • 语法：ip(选项)(参数)
      • 示例：ip link list //用ip命令显示网络设备的运行状态

    • hostname

      • 功能：用于显示和设置系统的主机名称，如果需要永久修改主机名，需要同时修改/etc/hosts和/etc/sysconfig/network的相关内容。
      • 语法：hostname(选项)(参数)
      • 示例：

    • ifconfig

      • 功能：被用于配置和显示Linux内核中网络接口的网络参数

      • 语法：ifconfig(参数)

      • 示例：

  • route

    • 功能：用来显示并设置Linux内核中的网络路由表，route命令设置的路由主要是静态路由

    • 语法：route(选项)(参数)

    • 示例：

  • ssh

    • 功能：是openssh套件中的客户端连接工具，可以给予ssh加密协议实现安全的远程登录服务器
    • 语法：ssh(选项)(参数)
    • 示例：ssh root@192.168.12.220 //以root用户连接192.168.12.220

  • firewall-cmd

    • 语法：host(选项)(参数)

    • 示例：

    • 语法：nslookup(选项)(参数)

    • 示例：

  • nc/netcat

    • 示例：

  • ping

    • 功能：用来测试主机之间网络的连通性

    • 语法：ping(选项)(参数)

    • 示例：

    • 功能：用于追踪数据包在网络上的传输时的全部路径，它默认发送的数据包大小是40字节

    • 语法：traceroute(选项)(参数)

    • rpm
      • 功能：是RPM软件包的管理工具
      • 语法：rpm(选项)(参数)
