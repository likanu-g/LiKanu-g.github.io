Java基本数据类型

数据类型分类

值类型(基本数据类型)

  • byte

    • 长度：8位，取值范围：-2^7~2^7-1

  • short

    • 长度：16位， 取值范围：-2^15~2^15-1

  • int

    • 长度：32位，取值范围：-2^31~2^31-1

  • long

    • 长度：64位，取值范围：-2^63~2^63-1，赋值时在值的后面加上l或者L

  • float

    • 长度：32位,赋值时在值的后面加上f或者F

  • double

    • 长度：64位,赋值时在值的后面加上d或者D

  • char

    • 长度：16位,存储unicode,用单引号复制

  • bolean

    • 长度：2位，赋值只能为true或者false中的一个

引用类型

  • 数组

  • 类 class

    • String是引用类型

  • 接口 interface

  • 枚举enum

值类型 VS 引用类型

  • 从概念上对比

    • 基本数据类型：变量名指向具体的数据
    • 引用数据类型：变量名指向存数据对象的内存地址

  • 从内存构建上对比

    • 基本数据类型：在变量声明后java虚拟机会立刻分配内存地址
    • 引用数据类型：在变量声明后不会立刻分配内存地址，而是将值存在内存中，用指针指向该值

  • 从使用方式上对比

    • 基本数据类型：使用时需要赋值，如果没有赋值，java虚拟机会初始化为默认值，判断两个变量相等用“==”操作符，如果相等返回true，否则返回false
    • 引用数据类型：使用时可以不赋值，或者赋值为null，但是如果没有赋值，虚拟机也不会初始化默认值，当要调用这个变量时会出现NullPointException
