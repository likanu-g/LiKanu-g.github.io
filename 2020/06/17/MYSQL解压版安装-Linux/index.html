MYSQL解压版安装(Linux)

 |
2020-06-17 (Updated: 2021-02-02)
数据库
MYSQL

1、删除先前安装的版本

  • 查找mysql安装目录

    ​ whereis mysql

  • 删除相关目录和文件

    ​ rm -rf /usr/lib64/mysql /usr/share/mysql

    ​

  • 验证是否删除完全

    ​ whereis mysql

    ​ find / -name mysql

    下面这个是我在docker中安装的mysql，这个无关紧要，并不影响我们现在的安装

  • 检查mysql用户组和用户是否存在，如果没有，则创建

2、下载MYSQL

下载方式有两种，一、在linux下用wget命令下载，二、在本地下载然后上传到linux上

[root@localhost /]# wget https://dev.mysql.com/get/Downloads/MySQL-5.7/mysql-5.7.24-linux-glibc2.12-x86_64.tar.gz

Windows下的下载就不赘述了，搜索引擎搜索MYSQL就会知道了

3、解压

将解压后的mysql-5.7.24-linux-glibc2.12-x86_64文件夹改名成mysql

然后将mysql复制(移动到)/usr/local，如果/usr/local已经有mysql文件夹则将已经存在的mysql文件夹改成其他名字,注意一定要放到这个目录下，不要改成其他的目录

4、安装前的准备

  • 在/usr/local/mysql目录下创建data目录

  • 创建my.cnf配置文件

  • 更改mysql目录下所有的目录及文件夹所属的用户组和用户，以及权限

  • 添加mysql的配置文件my.cnf /-JxlIaVs3ef

5、初始化安装设置

  • 进入mysql的bin目录

    成功初始化会生成临时密码

  • 启动mysql服务器

  • 添加软连接，并重启mysql服务

  • 登录mysql，修改密码(密码为步骤5生成的临时密码)

  • 开放远程连接

  • 设置开机自启

6、登录验证

使用数据库连接工具

连接成功

字符编码都正确
