Docker---2-Docker的基本使用

 |
2020-06-02 (Updated: 2020-08-09)
Docker
Linux

这一章主要讲解Docker的使用

主要分为以下几个部分：

  1. 在Docker中运行应用

    1. 使用命令docker run

      1. 写一个hello world,使用命令sudo docker run ubuntu:14.04 /bin/echo 'Hello world'

        ![写hello world](写hello world.png)

        显示我的docker本地中没有ubuntu:14.04镜像，然后从docker仓库中去pull拉取镜像

        ![写完hello world](写完hello world.png)

        然后hello world就写出来了

      2. 分析sudo docker run ubuntu:14.04 /bin/echo 'Hello world命令的执行过程，其实这个命令是三部分组成的，分别是docker run、ubuntu:14.04、/bin/echo 'Hello world大致分为以下步骤

        1. 寻找docker中的镜像ubuntu:14.04，如果没有则执行步骤2，如果有则执行步骤3

        2. 拉取镜像

        3. 执行/bin/echo 'Hello world，这个命令就是在终端中输出/bin/echo 'Hello world

          ![输出hello world](输出hello world.png)

  1. 使用容器

    使用容器有如下几个命令

    1. docker version 显示客户端和进程信息

      ![docker version](docker version.png)

    2. docker ps 列出在运行容器

      ![docker ps](docker ps.png)

    3. docker logs显示容器的标准输出

      ![docker logs](docker logs.png)

      docker logs 后面要跟一个参数CONTAINER ID

    4. docker stop 停止正在运行的容器

      ![ps stop](ps stop.png)

      和 docker logs一样 后面要跟一个参数CONTAINER ID

    5. docker rm删除容器，后面也要加一个容器的参数

  2. 使用docker镜像

    1. docker images列出所有镜像

    2. docker pull centos拉取Centos镜像

    3. 查找镜像

      1. 到 docker hub 的官方网站去搜索

        ![docker hub](docker hub.png)

      2. 使用docker search

        ![docker search](docker search.png)

    4. 实际运用

      docker 安装mysql

      1. 拉取镜像:docker pull mysql:5.7

      2. docker 启动mysql
