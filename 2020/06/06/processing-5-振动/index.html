角度与弧度：我们只需要知道360o = 2$\pi$弧度就够了

在processing中有角度转弧度的方法，processing 中旋转都是用弧度算的所以不要担心该怎么计算

注意以下的基本公式

位置 = 位置 + 速度

速度 = 速度 + 加速度

角度 = 角度 + 角速度

角速度 = 角速度 + 角加速度
