Elasticsearch

Elasticsearch 是一个分布式、可扩展、实时的搜索与数据分析引擎。 它能从项目一开始就赋予你的数据以搜索、分析和探索的能力，可用于实现全文搜索和实时数据统计。

Elasticsearch 安装和使用

  1. 下载Elasticsearch6.2.2的zip包，并解压到指定目录，下载地址：Elasticsearch
  2. 安装中文分词插件，在elasticsearch6.2.2\bin目录下执行elasticsearch-plugin install https://github.com/medcl/elasticsearch-analysis-ik/releases/download/v6.2.2/elasticsearch-analysis-ik-6.2.2.zip
  3. 运行bin目录下的elesticsearch.bat启动Elasticsearch
  4. 下载Kibana，作为访问Elasticsearch，下载地址：Kibana
  5. 运行bin目录下的kibana.bat，启动Kibana的用户界面
