覆盖Spring Boot自动配置

添加默认登录页面

在Spring Boot中添加添加默认登录界面很简单，我们只需要添加spring-boot-starter-security模块即可

添加gradle依赖

启动项目，查看console，可以看到如下启动日志

Using generated security password: 38524af3-eb4d-47e6-a352-f4468604096e为Spring Security自动生成的密码。

登录界面的用户名为user

在浏览器打开http://localhost:8080/，出现如下界面，这个时候我们就会有一个简单的登录界面

用户名和密码就在上面

自定义登录界面

上面的默认登录界面自然不是我们想要的，我们希望登录界面的用户名密码是存储数据库里的，还有自定义的也买了，当前页面也太简洁，完全达不到我们预期的效果。

要想自定义登录界面，我们只需要继承WebSecurityConfigurerAdapter然后重写configure()方法即可

通过配置文件外置配置

外部的基本配置基本都集中在application.yml文件中
