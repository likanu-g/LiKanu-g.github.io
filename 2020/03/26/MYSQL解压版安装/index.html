MYSQL解压版安装(Windows)

 |
2020-03-26 (Updated: 2021-01-24)
数据库
MYSQL

1、下载MYSQL

  • 下载地址https://dev.mysql.com/downloads/mysql/，选择自己想要的版本，本文以5.7.27为例

2、解压

  • 解压到任意一个一个没有中文名称的路径，我解压的是D:\Program Files路径下面，解压后mysql的路径为D:\Program Files\mysql-5.7.27-winx64

3、配置环境变量

  • 关于环境变量的配置，在哪里配置，就不赘述了，就在配置jdk环境变量的地方，配置方式也基本类似，新建一个MYSQL_HOME的环境变量然后将上面的mysql路径填在里面，然后再将MYSQL_HOME添加到path中，其实环境变量不配置也无所谓，主要是为了方便在cmd下直接使用mysql命令

5、安装初始化

  • 到前面四步准备工作已经基本完成，下面是安装部分的工作

  • 用管理员权限打开cmd,记住是管理员权限，这很重要，进入mysql解压的bin目录

  • 执行mysqld -install命令安装

  • 执行mysqld --initialize-insecure --user=mysql命令初始化

    由于我本机先安装了，这张图是网上找的

  • 完成后mysql的解压目录下会生成data文件夹，这个文件夹就是存放我们数据库数据的地方

    至此mysql解压版安装的流程已经基本结束，下面将说明mysql服务的启动以及密码设相关的问题

6、启动测试连接

  • 启动mysql服务，还是在mysql的bin目录下执行net start mysql，当然也可以在windows的服务管理界面手动启动服务

  • 设置root密码，执行mysqladmin -u root -p password,第一次执行这个命令会提示输入密码，直接回车就行，然后设置新的密码

    设置完成后就可以直接在cmd界面登录mysql测试

  • 登录mysql,执行mysql -u root -p,然后输入设置的命令

    mysql解压版的安装与测试已经全部完成
