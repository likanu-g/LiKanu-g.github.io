新安装IDEA的基本配置

 |
2020-05-29 (Updated: 2020-05-29)
java
Java

新安装的idea，我们常用的需要配置的项目有如下这些：

  • 字符编码 File Encodings

    设置的方式有两种

    1. 界面点击设置

      1. 在idea的主界面(工作区间)，点File–>Settings

      3. 在左上角的搜索框直接输入File Encodings，这种方式是最快的，如果你觉得打字比较慢，Edit–>File Encodings这样找也是一样的

      4. 然后将 Project Encoding he下面的Properties Files的编码改成UTF-8，idea默认是GBK的，至此字符设置完毕

    2. 快捷键设置

      • Ctrl+Alt+S进入设置界面
      • 下面的步骤同方法1中的步骤3,4

  • 忽略的文件 File Types

    1. 进入到设置界面(方法参见上面的字符编码设置)

    2. 在搜索栏输入搜索File Types，这时候搜索的结果是这样的，然而我们要设置的File Types

      其实最快的方式是这样，直接Editor–>File Types,下面两个都是一样的

    3. 点击File Types，进入设置界面

    4. 在下面红框出填写需要在项目中忽略显示的文件夹和文件，例如要忽略.idea文件夹就在末尾添加.idea;,要忽略.iml后缀的文件，就添加*.iml;,注意文件夹不需要加*，文件才需要

  • Maven的安装目录

    • 进入设置界面，搜索Maven

    • 如上图选择Maven的安装目录，勾选Override，选择配置文件路径，选择本地仓库路径

  • Gradle的安装目录

    • 进入设置界面搜索Gradle

    • 设置 gradle的安装目录

  • Git以及Github设置

    • 进入设置界面，搜索git

      这里可以手动设置git的安装目录或者idea自动探测

    • 接着上面的，点击GitHub，设置

    • 点击右边的+可以添加新的Github账号,输入账号密码登录即可
