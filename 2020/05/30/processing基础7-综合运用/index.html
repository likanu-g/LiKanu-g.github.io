processing

这个综合运用是我们将利用processing做一个电子钢琴APP，最终样式如图所示

在UI上不同位置点击，可以弹奏出不同的音乐，设计图稿如下

这个练习完成分为如下几个部分，篇幅问题，我将这综合运用的每一部分的代码放在后面给出

  1. 完成背景的绘制，效果图如下所示，这是设计图，每一部分的尺寸都给定了

    • processing中的绘图并不能绘制出，我们需要利用图片加载，大体分为三步

      1. 定义PImage全局变量

      2. 在setup()加载图片

      3. 在draw()中绘制图片

      不知道如何加载图片的可以在基础4–图片加载中学习相关的代码逻辑

  2. 完成按钮，和游标准线的绘制，以及按钮的逻辑功能

  3. 完成表格的点亮和取消点亮

  4. 完成亮块与准线的碰撞检测

  5. 完成音频的添加

  6. 整个练习的总结和联调

这6个部分的详细代码在后面分为6个部分给出，全部代码我已经放在github上

地址为：https://github.com/LiKanu/piano
